HDF5 is an efficient, well-documented, non-proprietary binary data format and library developed and maintained by the HDF Group. CISD has created a high-level API in Java for reading and writing HDF5 files, building on the libraries provided by the HDF Group.

In order to get started with the API, see the HDF5FactoryProvider class.

For simple standard use, have a look at IHDF5SimpleWriter interface for read-write access and the IHDF5SimpleReader interface for read-only access.

If you need to perform advanced operations, see the IHDF5Writer interface for read-write access and the IHDF5Reader interface for read-only access.
